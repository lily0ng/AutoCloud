# Docker Compose Examples

This directory contains various Docker Compose examples demonstrating different use cases and best practices.

## Examples Included:

1. **MERN Stack** (`mern-stack-compose.yml`)
   - MongoDB, Express.js, React, Node.js stack
   - Development environment with hot-reload
   - Volume persistence for database

2. **WordPress Development** (`wordpress-compose.yml`)
   - WordPress with MySQL
   - PHPMyAdmin for database management
   - Development optimized configuration

3. **Microservices Demo** (`microservices-compose.yml`)
   - Multiple interconnected services
   - Redis for caching
   - API Gateway pattern
   - Backend API and Frontend

4. **ELK Stack** (`elk-stack-compose.yml`)
   - Elasticsearch
   - Logstash
   - Kibana
   - Log monitoring setup

5. **Development Tools** (`devtools-compose.yml`)
   - PostgreSQL database
   - Adminer for database management
   - Redis cache
   - Mailhog for email testing
   - Traefik as reverse proxy

## Best Practices Implemented:

- Environment variable usage
- Volume management for persistence
- Network segregation
- Health checks
- Resource constraints
- Proper service naming
- Version control compatibility
- Security considerations
