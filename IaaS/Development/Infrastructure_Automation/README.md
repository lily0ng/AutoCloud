# Infrastructure Automation Project

This project contains automation scripts and configurations for:
- 4 OS Configurations
- 5 Network Configurations
- Infrastructure Service Automations (10 services with configurations)

## Directory Structure
```
Infrastructure_Automation/
├── os_automation/
├── network_automation/
├── service_automation/
└── config/
```

## Requirements
- Terraform >= 1.0.0
- Ansible >= 2.9
- Python >= 3.8
- AWS CLI >= 2.0
